# Impeccable Design Context

## Design Context

### Users
혼자 사용하는 개인 학습/일정 관리 앱. 주로 모바일에서 수시로 확인하고, 데스크탑에서도 사용.
과제 마감, 습관 추적, 공부 플랜을 빠르게 확인하는 것이 핵심 사용 패턴.

### Brand Personality
차분함 · 종이책 · 온기 (Calm · Analogue · Warm)

### Aesthetic Direction
- Claude 웹처럼 따뜻한 크림/갈색 계열 팔레트
- 종이책 같은 질감과 정제된 느낌
- AI틱하고 유치한 요소 완전 제거
- Light mode, warm cream background
- 이모지 없음, Lucide 아이콘만

### Design Principles
1. **Analogue First** — 화면이 아닌 종이 위에 인쇄된 것처럼 느껴져야 한다
2. **Quiet Confidence** — 화려하지 않지만 품질이 느껴지는 절제된 디자인
3. **Mobile Native** — 모바일에서 먼저 설계, 데스크탑은 확장
4. **Typographic Hierarchy** — 색보다 크기와 굵기로 위계를 만든다
5. **No AI Slop** — gradient text, glow, side-stripe 카드, 둥근 카드 반복 금지
